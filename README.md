# 📦 Estocaí

> **Sistema de gestão de estoque e frente de caixa para rotisserias.**

O **Estocaí** nasceu para resolver o caos do papel na **Brasinha Rotisseria**. O projeto transforma processos manuais, lentos e suscetíveis a falhas em uma operação digital ágil, focada no que realmente importa: o controle preciso do que entra e sai.

---

## 🚀 O Problema (O fim das comandas de papel)
Atualmente, a rotisserie enfrenta desafios que comprometem o tempo e o lucro:
* **Inventário Manual:** Perda de tempo conferindo estoque no papel antes de cada compra.
* **Fechamento Ineficaz:** A baixa de estoque é feita contando pilhas de comandas físicas após o expediente.
* **Canais Descentralizados:** Vendas por WhatsApp, balcão e reservas não possuem um controle unificado.
* **Falta de Visão Macro:** Sem dados digitais, é impossível prever altas e baixas de vendas para planejar compras.

## ✨ A Solução: Estocaí
O sistema unifica a operação em uma plataforma online:
1.  **Controle de Estoque Inteligente:** Saiba exatamente o que tem no freezer sem precisar abrir a porta.
2.  **Baixa Automática:** Ao fechar uma venda, o sistema atualiza o estoque instantaneamente.
3.  **Gestão de Vendas Multicanal:** Centralize pedidos presenciais e de WhatsApp em comandas digitais.
4.  **Cardápio Digital:** Exibição clara dos produtos e preços para a equipe e clientes.
5.  **Fidelidade Integrada:** Adeus cartões de papel! O histórico de compras do cliente gera benefícios automáticos.

## 🛠️ Tecnologias
Este projeto é desenvolvido como parte do TCC em **Análise e Desenvolvimento de Sistemas (Fatec Sorocaba)**, utilizando:
* Foco em performance e usabilidade.
* Arquitetura voltada para disponibilidade online.
* Código aberto (Open Source) para ajudar pequenos negócios locais.
